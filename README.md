# Zine Holders

![8 mini-comic zine holders in 2 zine holder cases, both old and current styles shown](/assets/image/zine_holders.jpeg)

A set of 3d printed cases for organizing zines. Available in mini-comic 8 page size and chick tracts size, and either case stacks on the other (although we recommend chick tracts on the bottom, as the case is slightly larger). Vertically half page size coming soon!

Print 4 holders and a holder case for a complete set. Can be printed with a 0.4mm nozzle and 0.2mm layer height, FDM, on a 130mm x 130mm print bed.

Reach out if you make a set or would like modifications.
